- Node.js 
- PostgreSQL
- Prisma

### Setup

```
npm install
```

### Lint

```
npm run lint
```

### Test

```
npm run test
```

### Run Development Server

```
npm run dev
```

## Prisma Setup

### Install Prisma

```
npm install -g prisma
```

### Deploy Prisma

```
prisma db push
```

### Generate Prisma Client

```
prisma generate
```

### Prisma Studio

```
prisma studio
```
